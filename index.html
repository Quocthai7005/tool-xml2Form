Welcome to the Kingan project

This project provides features related to XML and JSON structured data.

The features includes:

  • Transformation from XML into FORM, for better no-code experience with data modification.
  • Transformation from XML into JSON, for data migration purpose.
  • Transformation from JSON into FORM, for better no-code experience with data modification.
  • Transformation from JSON into XML, for data migration purpose.
